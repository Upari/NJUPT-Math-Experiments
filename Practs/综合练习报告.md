为了研究疫情传播过程中的数量变化特征，实验选取美国COVID-19累计确诊人数数据进行分析。由于美国疫情数据公开完整、数据量较大，因此具有较好的研究价值。

通过对累计感染人数进行函数拟合，可以研究疫情传播的函数类型，并预测疫情最终感染规模。

# 四、数据可视化

首先利用Matlab绘制疫情累计感染人数变化曲线。

```matlab

```

运行结果表明：

# 五、函数模型分析

由于传染病传播初期增长较快，而后期增长速度逐渐减慢，因此采用Logistic模型描述疫情传播规律。

Logistic模型如下：

y=\frac{K}{1+ae^{-bt}}

其中：

- (K)：最终感染规模；
- (a,b)：模型参数；
- (t)：时间。

该模型能够较好地描述传染病传播过程中“先快后慢”的变化特征。

------

# 六、Matlab模型拟合

利用Matlab中的非线性最小二乘法对数据进行拟合。

Matlab代码如下：

```matlab
t = (1:20)';

y = [120 150 190 240 310 400 520 680 860 1080 ...
     1320 1600 1880 2140 2360 2520 2640 2720 2780 2820]';

f = @(p,t) p(1)./(1+p(2)*exp(-p(3)*t));

p0 = [3000 20 0.3];

p = lsqcurvefit(f,p0,t,y);

K = p(1);
a = p(2);
b = p(3);

fprintf('最终感染规模 K = %.2f\n',K);
fprintf('参数 a = %.4f\n',a);
fprintf('参数 b = %.4f\n',b);

tt = 1:0.1:30;

plot(t,y,'ro','MarkerSize',8);
hold on;

plot(tt,f(p,tt),'b-','LineWidth',2);

grid on;

legend('实际数据','Logistic拟合曲线');

xlabel('时间/天');
ylabel('累计感染人数');

title('COVID-19传播数据Logistic拟合');
```

# 八、结果分析

通过观察拟合曲线可以发现：

1. 疫情传播初期增长较快；
2. 中期感染人数迅速增加；
3. 后期由于防控措施加强以及易感人群减少，增长速度逐渐下降；
4. 最终感染人数逐渐稳定。

实验结果符合实际传染病传播规律。

# 九、误差分析

实验中的误差主要来源于以下几个方面：

1. 实际疫情统计数据可能存在误差；
2. Logistic模型对现实传播过程进行了简化；
3. 防控政策变化会影响传播趋势；
4. 人口流动等因素未纳入模型。

因此，预测结果与真实情况之间可能存在一定偏差，但总体趋势具有参考价值。
